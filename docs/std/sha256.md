# std/sha256

## std/sha256

### `sha256`

```milo
pub fn sha256(input: &string): string
```

Compute the SHA-256 digest of a byte buffer, returned as 64-char lowercase hex.

### `sha256Bytes`

```milo
pub fn sha256Bytes(input: &string): string
```

Compute the SHA-256 digest of a byte buffer, returned as 32 raw bytes.
Use this when the digest feeds further bytes (HMAC, key derivation); `sha256`
wraps it for the common hex-string case.
